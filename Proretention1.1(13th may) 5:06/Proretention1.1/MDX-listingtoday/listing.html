The basics of Computer Networking

Submitted by HP Servers on 21-06-2013 02.05:50 (3,000 views)
Division Computer Networks

Of course, any network will need computers. If you're quite new in to the subject a good first lesson would be to attempt to connect two computers either by ethernet cable or wirelessly to ensure that they can share information. If it is possible to do this, you have created your first network.

However, once you add a lot more computers, printers, scanners and other peripherals, as well as an Web connection, things get considerably much more complicated. The average home network will have most or many of these components and business networks get infinitely much more complex. So, here is a simple overview of the hardware needed to build a network.

Wired or wireless?
The most standard requirement associated with a network is that the distinct components appear to be connected to each other. Here, there seem to be two choices. Wired or wireless. Ethernet cable continues to be the fastest of these two and remains an vital building block of most networks, especially huge business network where high data transfer speeds appear to be essential. That said, the subsequent generation of wireless is set to close the gap on ethernet cable with regards to speed, and offer a significantly better range. Already, wireless is favored in smaller networks simply because it spares the expense and the mess of ethernet cables, and is less difficult and cheaper to maintain.

Modems
Although networks can exist with out being connected in to the internet, It's rare in today's world. And to get a network to connect to the web a modem is required. Most modern home or small organization networks will work with a cable/ADSL modem. These frequently provide speeds of as much as 10 Mbps by ethernet. However, in reality, very few World wide web service providers can provide a connection that even comes close to this speed. Ethernet modems are the norm these days, although USB modems do exist and dial-up modems seem to be nonetheless used, though rarely for networks.

Routers
A router is vital for all networks as it offers the connection between the Local area Network (LAN), that is the home or little organization network, and also the Wide area Network (WAN), usually the internet. several home network will use a combined modem, router and switch that may allow the network to connect to the internet and allow any computers and peripherals on the network to communicate along with other each other. In quite big business networks the router (or sometimes a gateway is used) and switch will be individual hardware devices.

Switches and hubs
There is often several confusion concerning the difference between switches and hubs. Both allow computers and devices within a network to communicate with every other, but there are some significant differences between them. of the two, switches seem to be by far the better, and more expensive, option. Switches are important for larger networks due to the fact they make the transfer of Info a lot far more efficient.

When a switch receives data from a computer or peripheral it can determine Which device/s around the network around the network the data is intended for, and will only send it to its intended destination. This will not be noticed on a small home network but can produce a big difference on larger networks.

A hub is really an easier device that may connect anywhere between four and 24 devices. The data will pass throughout the hub but it the hub won't interfere With the data in any way. Therefore, data sent by means of a hub will be sent to, and may be accessed by, all devices on a network.

Firewalls
Hardware firewall devices aren't very necessary for home networks. The firewall software on your own individual computers' running systems' firewalls, as well as the security options provided by your router, ought to be much more than enough to protect your network.

However, for organization networks that are storing large amounts of sensitive Info a good hardware firewall is essential. All Info in one outside the network have to pass through this before reaching the network's significant switch. If properly configured by a network expert this should provide all the security a business needs. Further software firewall protection within the network is as well an option to secure the network further.

Computer network security is a subject that gives rise to concern among network owners and users. It's not enough that computers are inherently vulnerable to Trojans and viruses, now networks can be penetrated by hackers too for either fun or malicious reasons. Protecting your network is an essential responsibility and there are thankfully many uncomplicated ways to do that now

Security Risks Briefly Described There are many risks in a computer network, and most of them are consequences of ignorantly exposed virtual gateways for hackers. Some of them are described below

  • Operating a network can provide unethical persons a way to request or retrieve a file or resource they aren't authorized to access. If a hacker can subtly gain access to your network as an apparent user, they'll be able to retrieve any file or information allowable for viewing by users. The main problem lies in the fact that certain information accessed by users must not get into the hands of a hacker. In the event that the resources a hacker needs are confined within the admin dashboard, all he has to do is find a way to get admin privileges within the network

Introduction to Computer Network Architectures

Submitted by Zafer Imran on 21-06-2013 02.05:50 (1,000 views)
Division Computer Networks

Network architecture means design of computers, devices and media in a network. The computer network can be designed using different ways. The most popular network architectures are described below.

File-Server Network:
In File-Server computer network, a powerful computer having disk with large storage capacity and processing power is installed as central computer. This central computer is known as File-Server, Network Server, Application Server or simply a Server. A File-Server stores and manages files. The data files and software are stored on the Server. The individual computers on a network, called nodes, access the data files and software on the Server. It means that, the File-Server is used to store files and to forward them to nodes that request for them.

Client/Server Network:
Client/Server model is the most popular network model. It is also known as dedicated server model. In client/server network (or arrangement), a powerful computer is used as server. The server controls the functions of network. In addition to control the network, the server also provides a centralized storage area. The software and databases are stored on the server. Different nodes or computers connected to the network can access these software and databases.

All computers (other than computer server) connected in the network are called clients. The clients send requests to the server. Server receives the requests from clients and takes proper actions on these requests. It means that processing is usually done by the server. The results of processing are sent to the clients. Client/Server network may be LAN or WAN. For example, in a Client/Server network, a database is stored on the Server and the Client computers access the database. The Server portion of the DBMS (Database Management system) is stored on the Server that allows the Clients to add information in database or extract information from database. Similarly, the client portion of DBMS is stored on each Client computer through which request is sent to the Server to access the database. The Server processes the data and sends the result-to the Client computer.

Advantages:

  • The client/server network has the following advantages.
  • It reduces the volume of data traffic on the network.
  • It allows faster response at each node.
  • It allows using less expensive computers as clients because most of the work is done by server.
Disadvantages:
  • The main disadvantage of this network is that the entire network is affected if there is any problem in the server computer.

Peer-To-Peer Network:
In peer-to-peer (P2P) arrangement, all nodes (or computers) on the network have equal status. No one has control over others. It means that there is no central computer to control other computers on the network. Each computer stores files on its own storage devices and has its own peripheral devices. The users can share each other's data and devices (or resources) as and when needed. For example, a printer attached with any computer can be used by all computers connected in the network. Peer-to-Peer network is used in small offices or schools, where primary purpose of the network is to share storage and printers. A Peer-to-Peer network can also include a Server. In this case, a Peer-to-Peer local area network is similar to a File-Server network.

Advantages: The main advantages of peer-to-peer model are:

  • It is useful in small offices.
  • It is easy to design and to maintain.
Disadvantages:
    The main disadvantages of peer-to-peer model are:
  • It becomes slow under heavy use.
  • It does not provide the security to protect the data stored on computers connected in the network.
Hybrid Network Model:
The hybrid network has combined features of both client/server and peer-to-peer network (or arrangements) models. It also has a server. The users can share the data and software. Similarly, each node can store its own files, programs, and has its own peripheral devices. The users can also share each other's data and devices. These are the main advantage of hybrid arrangement.
